# القرآن الكريم

**Al-Quran Al-Kareem** — تطبيق قرآن كامل للقراءة والاستماع والتدبر.

## 📖 المميزات

- **قراءة القرآن** — عرض السور والآيات بخط عربي واضح مع تحكم بحجم الخط
- **الاستماع للآيات** — تشغيل تلاوة بصوت أشهر القراء (مشاري العفاسي، ياسر الدوسري، عبد الباسط، الحصري، سعد الغامدي، ماهر المعيقلي)
- **التشغيل التلقائي** — الانتقال التلقائي للآية التالية مع التمرير التلقائي وتمييز الآية النشطة
- **التفسير الميسر** — تفسير كل آية عبر API
- **الإشارات المرجعية** — حفظ الآيات للعودة إليها لاحقاً
- **الملاحظات** — إضافة ملاحظات على أي آية
- **النسخ والمشاركة** — نسخ الآيات ومشاركة السور
- **الوضع الليلي والنهاري** — واجهة مريحة للعين في الإضاءة المنخفضة
- **البحث عن السور** — بحث ذكي بالعربية والإنجليزية والأرقام
- **التصفية بالأجزاء** — عرض السور حسب أجزاء القرآن
- **حفظ آخر قراءة** — العودة تلقائياً لآخر سورة وآية تمت قراءتها
- **وضع عدم الاتصال** — النص يعمل بدون إنترنت

## 🛠 التقنيات المستخدمة

| التقنية | الاستخدام |
|---------|-----------|
| **Flutter** | إطار العمل الرئيسي |
| **Provider** | إدارة الحالة |
| **Google Fonts** | خط Amiri للعربية |
| **just_audio** | تشغيل التلاوة الصوتية |
| **shared_preferences** | تخزين الإعدادات والموقع والمحفوظات |
| **http** | جلب التفسير من API |
| **url_launcher** | فتح الروابط الخارجية |
| **fuzzy** | البحث الذكي عن السور |

## 📥 التثبيت

```bash
# استنساخ المشروع
git clone https://github.com/your-username/quran-app.git

# الدخول إلى المجلد
cd quran-app

# تثبيت الاعتماديات
flutter pub get

# تشغيل التطبيق
flutter run
```

### بناء APK

```bash
flutter build apk --release
```

## 🎨 الأيقونة

لإنشاء أيقونة التطبيق:

1. ضع صورة PNG (1024×1024) في `assets/icon/app_icon.png`
